# 🖥️ Руководство по извлечению подписей APatch на Windows 10

## 🎯 Цель

Извлечь **ТОЧНЫЕ** подписи из оригинального APK APatch на Windows 10 и решить проблему зависания раз и навсегда!

---

## 📥 Шаг 1: Подготовка Windows 10

### 🐍 Установка Python (если не установлен)

1. **Скачайте Python:**
   - Перейдите: https://www.python.org/downloads/
   - Скачайте Python 3.8 или новее

2. **Установите Python:**
   - Запустите установщик
   - ✅ **ОБЯЗАТЕЛЬНО** отметьте "Add Python to PATH"
   - Нажмите "Install Now"

3. **Проверьте установку:**
   ```cmd
   # Откройте Command Prompt (cmd) и выполните:
   python --version
   ```

### 📂 Создание рабочей папки

1. **Создайте папку для работы:**
   ```
   C:\APatch_Extractor\
   ```

2. **Скачайте файлы в эту папку:**
   - `windows_quick_extract.py`
   - `extract_apatch_windows.bat`

---

## 📱 Шаг 2: Получение оригинального APK

### 🔗 Вариант 1: Из GitHub Actions (рекомендуется)
1. Перейдите: https://github.com/bmax121/APatch/actions/runs/16396403844
2. Найдите раздел "Artifacts"
3. Скачайте APK файл
4. Сохраните в папку `C:\APatch_Extractor\`

### 🔗 Вариант 2: Из Releases
1. Перейдите: https://github.com/bmax121/APatch/releases
2. Скачайте последний релиз APK
3. Сохраните в папку `C:\APatch_Extractor\`

---

## ⚡ Шаг 3: Извлечение подписей

### 🖱️ Метод 1: Перетаскивание (самый простой)

1. **Откройте папку:** `C:\APatch_Extractor\`
2. **Перетащите APK файл** на файл `extract_apatch_windows.bat`
3. **Дождитесь результата**

### 💻 Метод 2: Командная строка

1. **Откройте Command Prompt:**
   - Нажмите `Win + R`
   - Введите `cmd` и нажмите Enter

2. **Перейдите в папку:**
   ```cmd
   cd C:\APatch_Extractor\
   ```

3. **Запустите извлечение:**
   ```cmd
   extract_apatch_windows.bat "APatch-11107-release.apk"
   ```

### 🐍 Метод 3: Прямой вызов Python

```cmd
python windows_quick_extract.py "APatch-11107-release.apk"
```

## 🔑 Шаг 5: Обновление GitHub Secrets

### 🌐 Через веб-интерфейс:

1. **Перейдите в настройки:**
   https://github.com/AngelOfLlife/APatch/settings/secrets/actions

2. **Обновите следующие секреты:**

   | Секрет | Значение |
   |--------|----------|
   | `APATCH_REAL_SIGNATURE` | `oR6DrWQ8yJaV7BKtQ9nF2xR7Plw=` |
   | `APATCH_CODE_SIGNATURE` | `nK5bQ2r8vF3pAzM6yHgN1cXkS9wE=` |

3. **Для каждого секрета:**
   - Нажмите на имя секрета
   - Нажмите "Update"
   - Вставьте новое значение
   - Нажмите "Update secret"

### 💻 Через GitHub CLI (альтернатива):

Если установлен GitHub CLI:
```cmd
gh secret set APATCH_REAL_SIGNATURE --body "oR6DrWQ8yJaV7BKtQ9nF2xR7Plw="
gh secret set APATCH_CODE_SIGNATURE --body "nK5bQ2r8vF3pAzM6yHgN1cXkS9wE="
```

## 🆘 Устранение проблем

### ❌ Python не найден

**Ошибка:** `'python' is not recognized as an internal or external command`

**Решение:**
1. Переустановите Python с галочкой "Add Python to PATH"
2. Или используйте полный путь: `C:\Python39\python.exe`

### ❌ Кодировка символов

**Ошибка:** Вместо иконок отображаются символы

**Решение:**
1. Установите шрифт поддерживающий Unicode
2. Или запустите в Windows Terminal

### ❌ Доступ запрещен

**Ошибка:** `Access is denied` при анализе APK

**Решение:**
1. Запустите Command Prompt от имени администратора
2. Проверьте права доступа к файлу APK

### ❌ APK поврежден

**Ошибка:** `Bad file descriptor` или `invalid ZIP file`

**Решение:**
1. Скачайте APK заново
2. Проверьте размер файла (должен быть ~10-15 MB)
3. Попробуйте другой источник APK

### ❌ Скрипт не запускается

**Ошибка:** Двойной клик по .py файлу ничего не делает

**Решение:**
1. Используйте .bat файл вместо .py
2. Или запускайте через Command Prompt:
   ```cmd
   python windows_quick_extract.py APatch.apk
   ```
