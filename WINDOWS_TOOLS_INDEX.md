## 🚀 **Быстрый старт для Windows 10:**

### **1. Скачайте файлы:**
```
📁 Скачайте папку windows_tools/ целиком
📱 Скачайте оригинальный APK APatch
```

### **2. Извлеките подписи:**
```cmd
# Перетащите APK на extract_apatch_windows.bat
# ИЛИ запустите:
python windows_quick_extract.py "APatch.apk"
```

### **3. Получите результат:**
```
APATCH_REAL_SIGNATURE=Uo6uAtr8EUXCnfCGs2ooeo3qJ9Y=
APATCH_CODE_SIGNATURE=w/FIHwoT4GORX42073W/q7jHzjgc5Zo+8EkPwFkY9Mc=
```

### **4. Обновите GitHub Secrets:**
https://github.com/AngelOfLlife/APatch/settings/secrets/actions

### **5. Запустите сборку → Получите рабочий APatch!** 🎉
